Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b.
Mas isso é um formato não semântico, não faz sentido para o HTML, o certo é utilizar o termo strong

Itálico / Ênfase

Da mesma forma, é possível criar um texto em itálico utilizando a tag i.
Ou então utilizando a tag certa que é semântica, o "em"

Texto Marcado

Podemos também criar um texto marcado utilizando a tag mark.

É possível colocar cada mark em cada cor que quiser, porém, também é possível mudar a cor padrão do marcador no campo head. Para isso, basta ir no campo head e digitar style, depois digitar mark e abrir colchetes e escrever backgroun-color e colocar a cor que quer.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno

Texto Deletado / Taxado

Podemos marcar um texto como excluído com a tag del, para indicar que ele deve ser lido, mas não considerado

Também é possível fazer isso com a tag s, porém ela está obsoleta.

Texto Inserido / Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Também se pode fazer isso com a tag u, para sublinhado, porém esta está obsoleta.

Texto Sobrescrito

Para inserir coisas como x2+3=0, utiliza-se a tag sup

Texto Subscrito

Para inserir coisas como H2O, utiliza-se a tag sub
